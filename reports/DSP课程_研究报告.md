# DSP课程研究报告

## 1. 研究背景

语音信号处理是数字信号处理的重要应用方向之一。与一般的音频信号相比，语音信号具有明确的声学结构、稳定的短时特征以及强烈的时间变化特性，因此非常适合作为 DSP 课程中的综合实验对象。

在语音合成领域，现代系统大量依赖深度学习模型，但这类系统往往结构复杂、参数不可解释、对课程实验不够友好。相比之下，经典的 Klatt 共振峰语音合成模型具有明显优势：

- 模型结构透明
- 参数物理意义明确
- 便于将频率、带宽、增益等 DSP 概念直接映射到听感变化
- 适合用于教学、实验分析和可解释性研究

因此，本研究选择以 Klatt 语音合成为基础，构建一个面向课程实验的可视化音节合成与分析系统。

## 2. 研究目标

本研究的目标不是构建高自然度的商业语音系统，而是建立一个可解释、可交互、可分析的 DSP 语音实验平台。具体目标包括：

1. 实现基于共振峰参数的语音合成系统
2. 建立音节级预设库，支持元音、双元音和辅音段建模
3. 设计交互式面板，支持音节拼接和试听
4. 提供课程报告所需的关键图像输出
5. 从时域、频域和结构参数角度分析合成语音

## 3. 理论基础

### 3.1 源-滤波器模型

人类语音可以抽象为“激励源 + 声道滤波器”的组合：

- 激励源代表声门振动或气流噪声
- 声道滤波器代表口腔、鼻腔等共振结构

在离散信号处理中，这种结构可表示为某个源信号经过一组线性滤波器后产生输出。Klatt 模型正是在这一思想基础上建立起来的。

### 3.2 共振峰与反共振

元音的主要音色由若干共振峰决定，共振峰本质上是频率响应中的峰值；鼻音和某些辅音还会引入反共振，即频率响应中的陷波结构。DSP 中可以通过二阶 IIR 共振器和 FIR 反共振器对其进行建模。

### 3.3 时变语音信号

语音并不是严格平稳的。一个音节内部，尤其是双元音和辅音-元音过渡阶段，频率参数会随时间变化。因此，在实验系统中使用多帧参数建模比单一静态帧更符合语音实际。

## 4. 研究方法

### 4.1 系统实现方法

本研究以 TypeScript 版 `klatt-syn` 为参考，重构出纯 Python 的语音合成引擎，并在其上扩展：

- JSON 音节预设管理
- PyQt5 音节拼接界面
- 300 DPI 图像输出
- 自动化测试与导出机制

研究采用“工程实现 + 听感验证 + 图像分析”的方法，而不是单纯停留在算法描述层面。

### 4.2 音节设计策略

为了兼顾实验可解释性和听觉可辨识性，当前预设分为三类：

1. 元音
   如 `a`, `i`, `u`, `e`, `o`, `ae`

2. 双元音
   如 `ai`, `ei`, `ou`

3. 辅音或辅音起始段
   如 `h`, `s`, `sh`, `f`, `m`, `n`，以及组合音节 `ha`, `hi`

这种设计支持两种研究路径：

- 直接研究某个音节内部的声学结构
- 通过手工拼接研究音节过渡与序列生成

### 4.3 参数选择方法

参数选择采用文献启发与经验调优结合的方式：

- 基频 `f0` 控制整体音高
- `oral_formant_freq` 和 `oral_formant_bw` 决定元音类别
- `noise` 激励用于 /h/、/s/、/sh/、/f/ 等无声段
- 双元音通过多帧 formant 插值实现平滑滑移
- 不同音节之间采用 `5 ms` crossfade 减少拼接点击声

该方法虽然不是自动学习方法，但在课程实验场景下具有较高可控性与解释性。

## 5. 实验设计

### 5.1 实验一：单音节声学结构观察

实验对象：`a`, `i`, `u`, `e`, `o`, `ae`

实验目的：

- 比较不同元音在波形、频谱和 formant 轨迹上的差异
- 观察 F1、F2 对音色变化的决定性影响

重点观察：

- `a` 具有较高 F1
- `i` 具有较低 F1 和较高 F2
- `u` 具有较低 F1、较低 F2
- `e`、`o` 位于中间元音区域

### 5.2 实验二：双元音过渡分析

实验对象：`ai`, `ei`, `ou`

实验目的：

- 观察多帧参数插值是否能形成连续元音滑移
- 验证 formant 轨迹图在解释双元音生成中的作用

重点观察：

- `ai` 的 F1 逐步下降、F2 逐步上升
- `ou` 的 F2 逐步下降并趋向后圆唇元音
- 双元音在声谱图中会表现为能量带位置连续移动

### 5.3 实验三：辅音与音节拼接

实验对象：`h`, `s`, `sh`, `f`, `m`, `n` 与若干元音拼接组合

实验目的：

- 观察不同辅音的噪声分布和时频特征
- 比较摩擦辅音与鼻音在波形和频谱上的区别
- 验证 crossfade 对拼接平滑性的影响

重点观察：

- `s` 和 `sh` 的高频能量分布不同
- `m`、`n` 体现鼻腔共振与反共振特性
- `h + vowel` 结构适合研究无声段向有声段的过渡

## 6. 结果分析思路

在课程报告中，建议从以下四个维度分析实验结果：

### 6.1 时域分析

波形图可用于观察：

- 音节总时长
- 无声段与有声段分布
- 拼接边界是否连续
- 不同辅音的包络差异

### 6.2 频域与时频分析

声谱图可用于观察：

- 元音能量集中区
- 摩擦辅音的高频噪声结构
- 双元音的连续频带迁移
- 鼻音的特殊共振模式

### 6.3 参数轨迹分析

共振峰轨迹图可用于证明：

- 元音差异主要源于 F1/F2 的不同组合
- 双元音是时间变化的 formant 序列
- 合成系统不是黑盒，而是受参数显式控制

### 6.4 结构与可解释性分析

时间轴图可以同时展示：

- 音节边界
- 激励类型变化
- 基频轨迹变化

这对于解释“为什么某段声音像辅音，为什么某段声音像元音”非常有效。

## 7. 研究意义

本研究的主要意义不在于追求高保真语音，而在于展示一个经典 DSP 语音系统如何被重新组织成现代课程实验平台。其意义体现在：

- 将语音源-滤波器模型具体化
- 将经典共振峰模型转化为可交互实验系统
- 将音频生成与图像分析统一起来
- 为 DSP 课程提供一个可重复、可解释、可扩展的研究案例

相比深度学习 TTS，本研究更强调模型透明度和实验可观察性，这一点尤其适合本科或研究生课程中的技术报告和实验展示。

## 8. 讨论

### 8.1 优势

- 参数明确，容易解释
- 能够直接联系 DSP 中的滤波器理论
- 可视化效果强，适合课程展示
- 交互成本低，支持快速拼接和试听

### 8.2 局限

- 参数主要依赖人工设计，缺少自动优化
- 合成语音可辨识但自然度有限
- 尚未实现面向句子级的韵律和音素协同控制
- 某些辅音仍属于近似建模，听感上与真实语音有差距

### 8.3 对课程实验的启发

该项目表明，在 DSP 课程中，实验系统不一定需要追求复杂模型。只要系统具备良好的可视化能力、可解释的参数结构和稳定的实验流程，就能成为高质量课程报告的支撑平台。

## 9. 后续研究方向

后续可以在以下方向继续研究：

- 基于文献或数据集标定更准确的英语元音参数
- 增加完整音节库与批量实验脚本
- 自动导出“音频 + 图像 + 参数表”的实验包
- 引入客观评价指标，如谱距离、谐噪比、能量包络
- 探索与传统 LPC 分析结合，用分析-合成一体化方法验证参数合理性
- 比较 Klatt 模型与现代神经语音模型在可解释性与自然度之间的取舍

## 10. 结论

本研究以经典 Klatt 语音合成模型为核心，构建了一个适用于 DSP 课程的音节级语音实验平台。研究结果表明：

- 基于共振峰参数的语音合成可以较好地表达元音、双元音和部分辅音特征
- 多帧参数插值能够有效描述双元音和辅音-元音过渡
- 将波形图、声谱图、共振峰轨迹图和时间轴图联合使用，能够显著提高实验分析的完整性和说服力
- 基于 PyQt5 的交互式系统有助于将研究过程从离线脚本推进到可视化、可演示、可复现的课程平台

因此，本项目不仅是一个工程实现，也可以视为一个关于“参数化语音合成在 DSP 教学中如何落地”的小型研究样例。它证明了经典方法在教学与研究场景中仍具有重要价值。
